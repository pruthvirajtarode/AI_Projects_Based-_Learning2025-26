# Mario Shooter — Play Store Submission Package

This folder contains all assets required for uploading your game "Mario Shooter" to Google Play Store.

## Included Files
1. **Mario-Shooter-Mobile-Enhanced.zip** — The complete Capacitor-ready game project (extract and build with Android Studio).
2. **icon_512x512.png** — Play Store app icon (512x512 px, PNG).
3. **feature_graphic_1024x500.png** — Play Store feature graphic banner (1024x500 px, PNG).
4. **screenshots.png** — Gameplay screenshots collage for store listing (upload individually or together).

## How to Use
1. Extract `Mario-Shooter-Mobile-Enhanced.zip` and follow the included README to generate your signed `.aab`.
2. On the Play Console, create a new app and fill in:
   - **App name:** Mario Shooter
   - **Category:** Game > Arcade / Action
   - **Description:** A fun, classic-style platform shooter game where Mario battles enemies with fireballs.
3. Upload these assets in their respective fields (icon, feature graphic, screenshots).
4. Complete the Content Rating, Data Safety, and Privacy Policy requirements.
5. Upload your `.aab` to the “Production” release and submit for review.

All assets are Play Store–ready and properly sized.
